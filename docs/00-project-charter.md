# 00. Project Charter

## Product Name

Multi-user Collaborative Trip Planner

## One-line Definition

A web app where multiple members collect places on a map, build itinerary plans,
and vote on the final trip proposal in real time.

## Target Users

- Friends planning a trip together
- Family travel groups
- Small teams organizing events with location + schedule constraints

## Core Problems

- Place preferences are scattered across chat tools
- No shared map + schedule workspace
- Hard to converge to one final plan fairly

## Success Metrics (MVP)

- 2-5 users can complete end-to-end workflow in one room:
  create room -> mark places -> create plans -> vote
- 95%+ marker and vote actions reflected to other members within 1 second
- At least one finalized winning plan generated per test room

## Non-goals (MVP)

- AI auto-planning
- Full offline mode
- Multi-language localization
- Enterprise org permissions
